API documentation

    • Users:-
      • Get
        • URL/api/users Disabled
            -to retrieve all users
        • URL/api/users/id Disabled
            -to retrieve user with _id=id
        • URL/api/users/email/
            -to retrieve user datausing jwt token sent as a query
        • URL/api/users/getemail
            -to retrieve user email using a jwt sent as a query
      • Post
        • URL/api/users
            -to add a user send the required fields in html body
        • URL/api/users/update
          • -to update a user's data, you can update
            • --name
            • --phone_no
            • --cart
            • --wishlist
            • --recently visited
            • --addresses
            • --email
        • URL/api/users/login
            check password and returns a jwt as token
      • Delete
        • URL/api/users/delete
            -to delete a user with email, send email in html body with fieldname email
